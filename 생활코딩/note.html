프론트엔드(Frontend)

프로그래밍의 한 분야. 웹사이트나 앱 등의 사용자 인터페이스(UI)를 개발하는 분야를 말한다. 프로그래밍에서의 프론트엔드라 함은 컴퓨터 소프트웨어에서 사용자와 상호작용하는 부분을 지칭한다.

HTML

HTML : 하이퍼 텍스트 마크업 언어 Hyper Text Markup Language의 약자 HTML은 웹 페이지 표시를 위해 개발된 지배적인 마크업 언어이다.

CSS

CSS : Cascading Style Sheet HTML 등의 마크업 언어로 작성된 문서가 실제로 웹사이트에 표현되는 방법을 정해주는 스타일 시트 언어.

Javascript

JavaScript : 자바스크립트는 객체지향 프로그래밍 언어이다.
이 언어는 웹 브라우저 내에서 주로 사용된다.
또한 다른 응용 프로그램의 내장 객체에도 접근할 수 있는 기능을 가지고 있다.

React Native

React Native : 자바스크립트는 객체지향 프로그래밍 언어이다.
이 언어는 웹 브라우저 내에서 주로 사용된다.
또한 다른 응용 프로그램의 내장 객체에도 접근할 수 있는 기능을 가지고 있다.
